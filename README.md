## 🚀 快速开始

### 前置条件

```bash
# 安装依赖
pip install edge-tts pillow

# 确保字体已安装
ls /usr/share/fonts/truetype/noto/NotoSansCJKsc-Bold.otf

# 确保ffmpeg已安装
which ffmpeg
```

### 一键生成

```bash
# 1. 生成帧
python3 generate.py

# 2. 生成配音
edge-tts --voice zh-CN-XiaoxiaoNeural --rate="+12%" --pitch="+8Hz" \
  --text "做课程太累了？...记得留个关注哟！" \
  --write-media OpenMAIC_配音.ogg

# 3. 合成视频
ffmpeg -r 30 -i v24_final_frames/frame_%04d.png \
  -i OpenMAIC_配音.ogg \
  -vf "subtitles=subtitle.srt:force_style='FontSize=20,MarginV=60'" \
  -pix_fmt yuv420p -c:v libx264 -crf 23 \
  -c:a libopus -b:a 128k -shortest \
  -y OpenMAIC_V24.7_最终版.mp4
```

## ⚠️ 常见失败原因

| 问题 | 原因 | 修复 |
|------|------|------|
| **字幕位置不对** | MarginV值设置错误 | MarginV=60（越小越靠底部） |
| **清华紫偏暗** | 色值不准确 | 严格使用 #9d4edd |
| **GitHub图标模糊** | 图标尺寸太小 | 放大至50px |
| **CTA转化弱** | 引导不明确 | "扫码上方二维码·立即领取" |
| **痛点不突出** | 卡片等高无重点 | 中间卡片放大4% |
