## 一、平台基础规范

### 1.1 尺寸与比例

| 平台 | 竖屏比例 | 分辨率 | 安全区域 |
|------|---------|--------|----------|
| **微信视频号** | 9:16 | 1080×1920 | 上下各120px |
| **抖音** | 9:16 | 1080×1920 | 下方150px |
| **快手** | 9:16 | 1080×1920 | 下方130px |


## 二、6大爆款模板类型

### 模板1: 痛点直击型

**适用**: 知识付费/教育/工具类产品
**黄金结构**: 痛点(3s) → 方案(7s) → 证明(5s) → CTA(5s)

```
┌─────────────────────────────┐
│ [品牌]    [信任背书]         │  ← 40px
├─────────────────────────────┤
│                             │
│     "你还在为XX烦恼？"       │  ← 主标题 72px
│      副标题(痛点放大)        │  ← 副标题 42px
│                             │
│   ┌────────┐ ┌────────┐    │
│   │ 痛点1   │ │ 痛点2   │    │  ← 痛点卡片
│   └────────┘ └────────┘    │
│                             │
│    "XX方法，彻底解决"        │  ← CTA暗示
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 痛点用红色/橙色高亮
- 痛点卡片中间放大10%
- 副标题用疑问句增强代入感

**参考案例**: 得到App、混沌学园课程推广

---

### 模板2: 数据对比型

**适用**: 产品优势展示/效率工具
**黄金结构**: Before(4s) → After(4s) → 对比(6s) → 转化(6s)

```
┌─────────────────────────────┐
│                             │
│      传统方式 vs AI方式       │  ← 48px 对比标题
│                             │
│   ┌──────────┐ ┌──────────┐ │
│   │  ❌ 2周  │ │  ✅ 5分钟 │ │  ← 左右对比卡片
│   │  ¥5000  │ │   免费   │ │
│   └──────────┘ └──────────┘ │
│                             │
│     效率提升 288 倍          │  ← 数据放大
│                             │
│   [立即体验按钮]             │  ← CTA
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 左右对比，差异用颜色区分(红vs绿)
- 数据用超大字号突出
- 对比线要有动态感

**参考案例**: Notion、飞书文档推广

---

### 模板3: 步骤流程型

**适用**: 教程/操作指南/使用说明
**黄金结构**: 结果预览(3s) → 步骤1-2-3(15s) → 成果展示(4s) → CTA(4s)

```
┌─────────────────────────────┐
│                             │
│    "3步学会XX技能"           │  ← 48px
│                             │
│   ╔══════╗                  │
│   ║ 01   ║ 输入你的需求      │  ← 步骤卡片
│   ╚══════╝                  │
│       ↓                     │
│   ╔══════╗                  │
│   ║ 02   ║ AI自动生成       │
│   ╚══════╝                  │
│       ↓                     │
│   ╔══════╗                  │
│   ║ 03   ║ 直接导出使用      │
│   ╚══════╝                  │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 步骤数字用圆形/胶囊背景
- 箭头要有流动感(渐变/动画)
- 每步配图标增强识别

**参考案例**: Canva、稿定设计教程

---

### 模板4: 知识图谱型

**适用**: 复杂概念解释/框架分享
**黄金结构**: 核心概念(5s) → 4象限展开(15s) → 总结(5s) → CTA(3s)

```
┌─────────────────────────────┐
│                             │
│        [核心概念]            │  ← 56px 居中
│                             │
│   ┌─────────┬─────────┐    │
│   │ 象限1   │ 象限2   │    │
│   │  标题   │  标题   │    │  ← 2×2网格
│   │  描述   │  描述   │    │
│   ├─────────┼─────────┤    │
│   │ 象限3   │ 象限4   │    │
│   │  标题   │  描述   │    │
│   └─────────┴─────────┘    │
│                             │
│      "完整版私信领取"        │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 象限用不同颜色区分
- 标题字号大于描述
- 网格线要细但清晰

**参考案例**: 混沌学园思维模型、得到知识城邦

---

### 模板5: 故事叙述型

**适用**: 个人IP/成长故事/案例分享
**黄金结构**: 场景(4s) → 冲突(6s) → 转折(6s) → 结局(6s) → CTA(4s)

```
┌─────────────────────────────┐
│                             │
│     第1幕: 困境             │
│     "我曾经也和你们一样..."  │
│                             │
│     [场景图片/图标]         │
│                             │
│     ↓ 转折点                │
│                             │
│     第2幕: 发现             │
│     "直到我遇到了XX..."     │
│                             │
│     [解决方案展示]          │
│                             │
│     第3幕: 结果             │
│     "现在我已经..."         │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 每幕用数字标识(01/02/03)
- 色彩随情绪变化(灰→亮)
- 留白要充足，不拥挤

**参考案例**: 刘润、梁宁等IP视频

### 模板6: 社交证明型

**适用**: 产品冷启动/信任建立/爆款打造
**黄金结构**: 数据冲击(5s) → 用户证言(10s) → 明星背书(5s) → CTA(6s)

```
┌─────────────────────────────┐
│                             │
│     200,000+ 人正在使用       │  ← 超大数字
│                             │
│   ┌─────────────────────┐   │
│   │ "这是我用过最好的XX" │   │  ← 用户证言卡片
│   │    ⭐⭐⭐⭐⭐          │   │
│   │    - 用户名          │   │
│   └─────────────────────┘   │
│                             │
│   [用户头像1] [头像2] [头像3] │
│                             │
│   "你也想加入吗？"          │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 数字用动画/渐变效果
- 证言要有头像+名字增加真实感
- 用户头像用宫格排列

**参考案例**: 小红书种草、抖音好物推荐

## 三、排版原子规范

### 3.1 字号系统

| 层级 | 用途 | 视频号 | 抖音 | 快手 |
|------|------|--------|------|------|
| **超大标题** | 主标题、数字 | 72px | 76px | 68px |
| **大标题** | 副标题、CTA | 48px | 52px | 46px |
| **中标题** | 卡片标题 | 32px | 36px | 30px |
| **正文** | 描述、说明 | 26px | 28px | 24px |
| **小字** | 标签、提示 | 20px | 22px | 18px |
| **微字** | 版权、来源 | 16px | 18px | 14px |

### 3.2 间距系统

```python
# 8px基倍数系统
SPACE_UNIT = 8

MARGIN_SMALL = 8      # 紧凑间距
MARGIN_NORMAL = 16    # 标准间距
MARGIN_MEDIUM = 24    # 中等间距
MARGIN_LARGE = 32     # 大间距
MARGIN_XLARGE = 48    # 超大间距

# 卡片间距
CARD_GAP = 24
CARD_PADDING = 32
CARD_RADIUS = 16

# 安全边距
SAFE_TOP = 120
SAFE_BOTTOM = 120
SAFE_HORIZONTAL = 80
```

### 3.3 色彩对比度

| 背景色 | 文字色 | 对比度 | 等级 |
|--------|--------|--------|------|
| #0D1117 | #FFFFFF | 15.3:1 | ✅ AAA |
| #0D1117 | #58A6FF | 8.2:1 | ✅ AA |
| #161B22 | #FFFFFF | 13.1:1 | ✅ AAA |
| #161B22 | #8B949E | 4.5:1 | ✅ AA |

## 五、爆款元素清单

### 5.1 必须包含
- [ ] 前3秒有冲击力的标题
- [ ] 品牌/个人标识在顶部
- [ ] 至少1个视觉锚点(放大/高亮)
- [ ] 明确的CTA引导
- [ ] 倒计时/稀缺性暗示

### 5.2 强烈建议
- [ ] 数字数据可视化
- [ ] 色彩情绪递进(灰→亮)
- [ ] 图标+文字双重编码
- [ ] 社交证明元素
- [ ] 底部二维码/关注引导

### 5.3 避免踩坑
- [ ] 文字过多过密
- [ ] 颜色超过5种主色
- [ ] 无安全边距贴边
- [ ] CTA不突出或缺失
- [ ] 动线混乱无重点
