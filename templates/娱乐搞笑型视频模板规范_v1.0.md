## 一、娱乐搞笑设计 DNA

### 1.1 核心视觉元素

| 元素 | 规范 | 示例 |
|------|------|------|
| **爆炸框** | 锯齿边框，动态抖动 | 笑点强调 |
| **表情包** | 超大头，夸张表情 | 情绪放大 |
| **拟人化** | 物品加表情/手脚 | 趣味动画 |
| **弹幕** | 斜向飞过，多种颜色 | 吐槽/梗 |
| **特效字** | 弹跳、旋转、放大 | 金句/梗 |
| **贴纸** | 可爱、搞怪、流行 | 装饰点缀 |

## 二、娱乐搞笑6大模板

### 模板F1: 吐槽梗图型

**适用**: 热点吐槽、生活段子、搞笑对比
**时长**: 15-20秒
**结构**: 铺垫(3s) → 期待(4s) → 反转(5s) → 笑点(5s) → CTA(3s)

```
┌─────────────────────────────┐
│                             │
│   当你以为...               │  ← 铺垫
│                             │
│   [正常场景图]              │
│   周末可以好好休息 😊       │
│                             │
│   vs 实际上...              │  ← 反转提示
│                             │
│   💥 爆炸效果 💥            │
│                             │
│   [反转场景图]              │
│   老板: "加个班" 📱         │
│        ↓                    │
│   [表情包: 崩溃大哭]        │  ← 笑点
│                             │
│   弹幕飞过:                 │
│   "太真实了" "人间真实"     │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- "当你以为...vs实际上"经典结构
- 反转要有爆炸框效果
- 表情包要夸张到位
- 弹幕增加互动感

**参考**: 暴走漫画、段子手本人

### 模板F2: 魔性洗脑型

**适用**: 神曲改编、节奏向、鬼畜内容
**时长**: 15-20秒
**结构**: Hook(2s) → 主歌(8s) → 副歌(6s) → 重复(4s) → CTA(3s)

```
┌─────────────────────────────┐
│                             │
│   🎵 BGM起                 │
│                             │
│   [魔性音乐可视化]          │
│   波形图跳动               │
│                             │
│   歌词大字弹出:             │
│   "打工人 打工魂"           │
│      ↓ 弹跳效果            │
│   "打工都是人上人"          │
│      ↓ 旋转效果            │
│   "月入一千八"              │
│      ↓ 缩小效果            │
│   "每天笑哈哈"              │
│      ↓ 放大效果            │
│                             │
│   [循环重复+加速]          │
│                             │
│   结尾定格: 😎              │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 歌词要随节奏弹跳
- 每个词不同动效
- 颜色要鲜艳高饱和
- 结尾要有魔性定格

**参考**: 朱一旦的枯燥生活、手工耿

**结构**: 场景(3s) → 对话1(6s) → 对话2(6s) → 冲突(6s) → 笑点(6s) → CTA(4s)

```
┌─────────────────────────────┐
│                             │
│   📍 场景: 办公室          │  ← 场景标签
│                             │
│   ┌─────────────────────┐   │
│   │ 👤 老板             │   │
│   │ "方案做得怎么样？"  │   │  ← 对话框
│   └─────────────────────┘   │
│                             │
│   ┌─────────────────────┐   │
│   │ 👤 我               │   │
│   │ "快好了快好了"      │   │
│   │ (心虚表情)          │   │
│   └─────────────────────┘   │
│                             │
│   ┌─────────────────────┐   │
│   │ 👤 老板             │   │
│   │ "昨天也是这么说的"  │   │
│   └─────────────────────┘   │
│        ↓                    │
│   [表情包: 尴尬而不失礼貌]  │  ← 笑点
│                             │
│   #职场 #搞笑 #真实         │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 对话框要有头像
- 表情要配合文字
- 场景标签要醒目
- 结尾要有话题标签

**参考**: papi酱、办公室小野

## 三、娱乐风动效规范

### 3.1 搞笑动效

| 动效 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| **弹跳** | 600ms | ease-out-bounce | 笑点强调 |
| **抖动** | 200ms | linear | 震惊/崩溃 |
| **旋转** | 800ms | ease-in-out | 眩晕/迷惑 |
| **放大** | 400ms | ease-out | 重点强调 |
| **缩小** | 300ms | ease-in | 失望/尴尬 |
| **飞入** | 500ms | ease-out | 弹幕/评论 |
| **震动** | 100ms×3 | linear | 强调/震惊 |

### 3.2 音效配合

| 音效 | 使用场景 | 时长 |
|------|---------|------|
| **"叮"** | 反转/顿悟 | 0.5s |
| **"哐"** | 摔倒/失败 | 0.3s |
| **"哇哦"** | 惊讶/赞叹 | 0.8s |
| **笑声** | 笑点之后 | 1-2s |
| ** drumroll** | 悬念/揭晓 | 2-3s |
| **"啪"** | 打脸/反转 | 0.2s |

## 四、字体与排版

### 4.1 推荐字体

| 用途 | 字体 | 效果 |

| **标题** | 站酷快乐体, 优设标题黑 | 活泼有趣 |
