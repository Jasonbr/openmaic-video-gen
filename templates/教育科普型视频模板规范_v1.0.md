## 二、教育科普6大模板

### 模板E1: 知识点拆解型

**适用**: 复杂概念讲解、公式推导、原理解析
**时长**: 25-30秒
**结构**: 引入(4s) → 概念(6s) → 拆解(10s) → 总结(6s) → CTA(4s)

```
┌─────────────────────────────┐
│                             │
│   📚 今天学习: 复利效应      │  ← 课程标题
│                             │
│   ┌─────────────────────┐   │
│   │                     │   │
│   │    复利公式         │   │  ← 公式展示
│   │    A = P(1+r)^t    │   │
│   │                     │   │
│   └─────────────────────┘   │
│                             │
│   拆解3要素:                │
│   ┌────────┬────────┐      │
│   │ 本金P  │ 利率r  │      │  ← 3要素卡片
│   ├────────┼────────┤      │
│   │ 时间t  │        │      │
│   └────────┴────────┘      │
│                             │
│   💡 关键洞察:              │
│   "时间是最重要变量"        │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 公式要有手写动画效果
- 重点变量用颜色标注
- 拆解步骤要编号(01/02/03)
- 最终要有"啊哈时刻"

**参考**: 李永乐老师、薛兆丰经济学

---

### 模板E2: 故事案例型

**适用**: 历史故事、案例分析、人物传记
**时长**: 30-35秒
**结构**: 背景(5s) → 冲突(8s) → 转折(8s) → 启示(8s) → CTA(6s)

```
┌─────────────────────────────┐
│                             │
│   📖 牛顿与苹果的故事         │  ← 故事标题
│                             │
│   ╔═══════════════════════╗ │
│   ║  1666年 · 英格兰      ║ │  ← 时间地点
│   ╚═══════════════════════╝ │
│                             │
│   ┌─────────────────────┐   │
│   │ [场景插画]          │   │  ← 故事场景
│   │  年轻的牛顿         │   │
│   │  坐在苹果树下       │   │
│   └─────────────────────┘   │
│                             │
│   🍎 一个苹果落下...        │  ← 关键事件
│                             │
│   💡 启发思考:              │
│   "为什么苹果向下？"        │
│                             │
│   → 万有引力定律诞生        │  ← 结论
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 时间地点用复古边框
- 关键场景要有插画感
- 人物要有Q版形象
- 转折处要有"叮"的音效提示

**参考**: 混知、飞碟说

### 模板E3: 对比辨析型

**适用**: 易混淆概念、相近知识、正反案例
**时长**: 25-30秒
**结构**: 引入(3s) → 概念A(6s) → 概念B(6s) → 对比(8s) → CTA(6s)

```
┌─────────────────────────────┐
│                             │
│   🆚 容易混淆的两个概念      │  ← 对比标题
│                             │
│   ┌──────────┬──────────┐ │
│   │           │           │ │
│   │  效率     │   效果   │ │  ← 双栏对比
│   │           │           │ │
│   │ 做事情的  │  做事情  │ │
│   │  速度     │   质量   │ │
│   │           │           │ │
│   │  ⚡️      │    ⭐    │ │
│   │ 快不快    │  好不好  │ │
│   │           │           │ │
│   └──────────┴──────────┘ │
│                             │
│   关键区别:                 │
│   效率 × 效果 = 效能        │
│                             │
│   [案例说明]                │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 左右对比要分明
- 核心差异用箭头指示
- 用图标强化概念(⚡️vs⭐)
- 最后要有记忆口诀

**参考**: 得到App、混沌学园

### 模板E4: 时间线型

**适用**: 历史演变、发展进程、阶段讲解
**时长**: 30-35秒
**结构**: 起点(4s) → 阶段1(6s) → 阶段2(6s) → 阶段3(6s) → 现状(6s) → CTA(7s)

```
┌─────────────────────────────┐
│                             │
│   📅 人工智能发展历程        │  ← 主题
│                             │
│   ━━━━━━━━━━━━━━━━━━━━━━   │  ← 时间轴线
│                             │
│   1950 ┃                    │
│   ━━━━━╋━━━━ 图灵测试       │  ← 里程碑1
│        ┃                    │
│   1980 ┃                    │
│   ━━━━━╋━━━━ 专家系统       │  ← 里程碑2
│        ┃                    │
│   2010 ┃                    │
│   ━━━━━╋━━━━ 深度学习       │  ← 里程碑3
│        ┃                    │
│   2023 ┃                    │
│   ━━━━━╋━━━━ ChatGPT时代     │  ← 里程碑4
│        ┃                    │
│        ↓                    │
│   🚀 未来已来               │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 时间轴要有进度感
- 每个节点要有年份+事件
- 当前节点要突出
- 可以用火车/火箭动画

**参考**: 回形针、混乱博物馆

### 模板E5: 思维导图型

**适用**: 知识体系、框架模型、概念网络
**时长**: 25-30秒
**结构**: 中心概念(5s) → 一级分支(8s) → 二级分支(8s) → 应用(6s) → CTA(5s)

```
┌─────────────────────────────┐
│                             │
│         学习力              │  ← 中心概念
│            ┃                │
│      ┌─────┼─────┐          │
│      ┃     ┃     ┃          │
│   ┌──┴──┐ ┌┴┐  ┌┴┐          │
│   │动力│ │力│  │力│          │  ← 分支展开
│   └─────┘ │法│  │习│          │
│           │   │  │惯│          │
│           └───┘  └───┘          │
│                             │
│   3个维度 × 3个方法 = 9种   │  ← 总结
│                             │
│   [实际应用案例]            │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 中心概念要最大最突出
- 分支要有动画展开效果
- 不同层级用不同颜色
- 最后要有完整版图

**参考**: MindMaster、XMind推广

### 模板E6: 实验演示型

**适用**: 科学原理、物理化学、动手实验
**时长**: 30-35秒
**结构**: 问题(4s) → 材料(5s) → 步骤(12s) → 结果(7s) → 原理(7s) → CTA(5s)

```
┌─────────────────────────────┐
│                             │
│   🔬 科学小实验              │
│   杯子里的水为什么不洒？      │  ← 问题
│                             │
│   📝 材料清单:               │
│   [杯子] [纸片] [水]       │  ← 材料图标
│                             │
│   📋 步骤:                   │
│   ① 杯子装满水               │
│      [步骤图1]               │
│   ② 用纸片盖住杯口           │
│      [步骤图2] → 动画       │
│   ③ 快速翻转                 │
│      [步骤图3] → 慢动作     │
│                             │
│   ✨ 结果: 水没洒！          │  ← 结果展示
│                             │
│   💡 原理: 大气压强          │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 材料要有实物图标
- 步骤要有编号和箭头
- 关键动作要慢镜头
- 原理要配示意图

**参考**: 这不科学啊、模型师老原儿

## 三、教育风动效规范

### 3.1 推荐动效

| 动效 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| **书写** | 100ms/字 | ease-out | 公式/标题 |
| **翻页** | 600ms | ease-in-out | 知识点切换 |
| **高亮** | 400ms | ease-out | 重点标记 |
| **思维导图展开** | 800ms | ease-out | 知识框架 |
| **时间线流动** | 连续 | linear | 历史演变 |
| **实验慢动作** | 2-3s | linear | 关键步骤 |

### 3.2 视觉强调

```css
/* 重点标记 - 荧光笔效果 */
.highlight {
    background: linear-gradient(120deg, transparent 0%, #FDE047 0%, #FDE047 100%, transparent 100%);
    background-size: 100% 40%;
    background-position: 0 88%;
    background-repeat: no-repeat;
}

/* 粉笔手写 */
.chalk-text {
    font-family: 'Comic Sans MS', cursive;
    text-shadow: 2px 2px 0px rgba(0,0,0,0.1);
}

/* 纸张纹理 */
.paper-texture {
    background-image: url('paper-grain.png');
    background-color: #FEF3C7;
}
```

## 四、字体与排版

### 4.1 推荐字体

| 用途 | 中文字体 | 英文字体 |
|------|---------|---------|
| **标题** | 站酷庆科黄油体, 优设标题黑 | Poppins, Montserrat |
| **正文** | 思源黑体, 苹方 | Inter, Open Sans |
| **手写** | 站酷快乐体, 徐静蕾手写体 | Comic Neue, Caveat |
| **公式** | Latin Modern Math | KaTeX_Main |

### 4.2 排版规范

```python
EDUCATION_LAYOUT = {
    'title_size': 56,        # 课程标题
    'section_size': 40,      # 章节标题
    'body_size': 28,         # 正文
    'caption_size': 22,      # 说明文字
    
    'line_spacing': 1.6,     # 行高
    'paragraph_spacing': 32, # 段间距
    'card_padding': 36,      # 卡片内边距
    'border_radius': 16,     # 圆角
}
```
