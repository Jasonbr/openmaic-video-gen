## 一、科技风设计 DNA

### 1.1 核心视觉元素

| 元素 | 规范 | 示例 |
|------|------|------|
| **网格背景** | 0.5px线条，10-20px间距 | GitHub Dark网格 |
| **霓虹光效** | 外发光3-5px，内发光1-2px | 按钮/标题 |

| **终端窗口** | 圆角8px，$提示符 | 命令行演示 |
| **数据流动** | 粒子/线条动画 | API调用可视化 |
| ** glassmorphism** | 透明度0.7-0.9，模糊10-20px | 卡片背景 |

## 二、科技风6大模板

### 模板T1: 代码演示型

**适用**: 开发者工具、CLI工具、API产品
**时长**: 20-25秒
**结构**: 问题代码(5s) → 解决方案(8s) → 执行结果(7s) → CTA(5s)

```
┌─────────────────────────────┐
│  $ terminal — 80×24         │  ← 终端标题栏
├─────────────────────────────┤
│                             │
│  $ 传统方式:                │
│  > 200行代码...            │  ← 红色错误代码
│  > Error: Timeout          │
│                             │
│     ↓ 一键优化 ↓           │  ← 霓虹箭头
│                             │
│  $ OpenMAIC:               │
│  > generate_course()       │  ← 绿色成功代码
│  > ✓ Done in 5s            │
│                             │
│  ┌─────────────────────┐   │
│  │  生成结果预览        │   │  ← 结果卡片
│  └─────────────────────┘   │
│                             │
└─────────────────────────────┘
```

**设计要点**:
- 终端窗口要有$提示符和光标闪烁
- 错误代码用红色，成功用绿色
- 代码行号要显示
- 滚动效果模拟真实终端

**参考**: Warp Terminal、Fig、GitHub CLI推广视频

## 四、科技风动效规范

### 4.1 推荐动效

| 动效 | 时长 | 缓动 | 用途 |
|------|------|------|------|
| **打字机** | 50ms/字 | linear | 代码输入 |
| **渐入** | 300ms | ease-out | 卡片出现 |
| **霓虹闪烁** | 2s循环 | ease-in-out | 强调元素 |
| **粒子流动** | 连续 | linear | 背景装饰 |
| **数字滚动** | 800ms | ease-out | 数据展示 |
| **进度填充** | 1.5s | ease-in-out | 进度条 |

### 4.2 光效参数

```css
/* 霓虹发光 */
.neon-glow {
    box-shadow: 
        0 0 5px currentColor,
        0 0 10px currentColor,
        0 0 20px currentColor;
}

/* 内发光 */
.inner-glow {
    box-shadow: inset 0 0 10px rgba(88, 166, 255, 0.3);
}

/* 扫描线 */
.scanline {
    background: linear-gradient(
        to bottom,
        transparent 50%,
        rgba(0, 0, 0, 0.1) 50%
    );
    background-size: 100% 4px;
}
```

## 五、科技风 vs 通用模板对比

| 维度 | 通用模板 | 科技风模板 |
|------|---------|-----------|
| **背景** | 纯色/渐变 | 网格+深色 |
| **强调** | 暖色/金色 | 霓虹冷色 |
| **字体** | 圆润/友好 | 等宽/锐利 |
| **元素** | 图标/插画 | 代码/数据 |
| **动效** | 平滑/自然 | 机械/精准 |
| **情感** | 温暖/亲和 | 专业/未来 |
