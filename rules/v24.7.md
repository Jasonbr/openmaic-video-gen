## 十、常见问题修复

### Q1: 字幕位置不对
- **问题**: 字幕太靠上或太靠下
- **解决**: MarginV值调整（越小越靠底部）
  - 太靠上: 减小MarginV (400→280→60)
  - 太靠下: 增大MarginV

### Q2: 清华紫颜色不对
- **问题**: 紫色偏暗或偏亮
- **解决**: 严格使用 #9d4edd (157, 78, 221)

### Q3: GitHub图标不清晰
- **问题**: Octocat图标太小
- **解决**: 图标尺寸50px（放大20%）

### Q4: CTA转化力不足
- **问题**: 用户不知道下一步
- **解决**: 
  1. 底部CTA直接指向二维码
  2. 3步流程可视化
  3. "回复【课件】自动领取"明确闭环

## 十二、生成命令模板

```bash
# 1. 生成帧
cd /root/OpenMAIC宣传视频
python3 create_v24.py

# 2. 创建字幕文件
cat > OpenMAIC_V24_字幕.srt << 'EOF'
[字幕内容见第五节]
EOF

# 3. 合成视频（带字幕）
ffmpeg -r 30 -i v24_final_frames/frame_%04d.png \
  -i OpenMAIC_晓晓配音.ogg \
  -vf "subtitles=OpenMAIC_V24_字幕.srt:fontsdir=/usr/share/fonts/truetype/noto:force_style='FontName=Noto Sans CJK SC Bold,FontSize=20,PrimaryColour=&H00FFFFFF,OutlineColour=&H00000000,Outline=2,Alignment=2,MarginV=60'" \
  -pix_fmt yuv420p -c:v libx264 -crf 23 \
  -c:a libopus -b:a 128k -shortest \
  -y OpenMAIC_V24_最终版.mp4
```
